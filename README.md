## Install

```sh
yarn add --dev --exact stylelint @ymkz/stylelint-config
```

## Usage

Write the following code in `.stylelintrc.json`

```
{
  "extends": ["@ymkz/stylelint-config"]
}
```

## Requirements

- `stylelint` >= `12.0.1`
